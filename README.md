## Project setup <br>

### Clone the repository
```
git clone https://github.com/mateusdayrell/t6-prog-II.git
```

### Access the project folder
```
cd t6-mateus-dayrell
```


### Install the dependencies
```
- BACKEND
# cd backend
# npm install
```

```
- FRONTEND
# cd frontend
# npm install
```


### Create a database named 'testdb'

### Serve the aplication
```
- BACKEND
# cd backend
# node server.js
```

```
- FRONTEND
# cd frontend
# npm start
```
